# Remind Thyself (MVP)

Android 11+ приложение с ежедневными напоминаниями.

## Что реализовано
- Управление списком напоминаний (до 10): текст, время, вкл/выкл, удаление.
- Ежедневное повторение каждого активного напоминания.
- Случайный сдвиг времени срабатывания от -5 до +5 минут.
- Уведомление с текстом напоминания, остаётся в шторке до свайпа.
- Один общий звук для всех напоминаний.
- Выбор и предпрослушивание звука на экране настроек.
- Проверка системных настроек: уведомления, exact alarms, battery optimization.
- Перепланирование после перезагрузки и смены времени/таймзоны.

## Сборка
```bash
JAVA_HOME=/path/to/jdk17 gradle :app:assembleDebug
```

## Установка
Установите APK вручную и в экране настроек выдайте/проверьте нужные разрешения.
